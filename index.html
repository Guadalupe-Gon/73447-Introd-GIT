Comandos básicos de GIT

  • git init
  • git add .
  • git commit -m "Mensaje"
  • git status
  • git log
  • git branch
  • git checkout -b "Nombre de la rama"
  • git checkout "Nombre de la rama"
  • git merge "Nombre de la rama"
  • git push origin "Nombre de la rama"
Utilidades GIT

Como iniciar un proyecto GIT

  1. Crear un repositorio GIT

    git init

  2. Agregar archivos al repositorio (tomar foto del código)

    git add .

  3. Generar una nueva versión de nuestro proyecto (commit)

    git commit -m "Mensaje del commit"

  4. Ver el historial de cambios (GitGraph)

    git log

  5. Modificar nombre de la rama

    git brunch -m nombre-rama

  6. Vincular mi repositorio en mi proyecto GIT con un repositorio en GitHub

    git remote add origin http://url-repositorio.git

  7. Subir mi proyecto a GitHub

    git push -u origin nombre-rama
